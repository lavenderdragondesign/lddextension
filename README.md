## 📄 Flagship Feature — Branded PDF Builder

The biggest addition in the rebuild.

A **Canva-style PDF builder** that:

- Reads the download link from your original MD PDF  
- Embeds it into a custom button in your new design  
- Lets you fully brand your delivery file  

### What you can do:
- Drag & drop layout editing  
- Add logos, text, images, shapes  
- Control full layout + sizing  
- Export as PDF / PNG / ZIP  
- Keep everything fully browser-side  

👉 Your PDF becomes part of your brand, not just a file.

## 📦 Install (under a minute)

Until Chrome Web Store approval:

1. Download this release ZIP  
2. Extract it somewhere permanent  
3. Open `Manage Extensions`  
4. Enable **Developer Mode**  
5. Click **Load Unpacked**
6. Pin to toolbar
7. Select the folder  

Open MyDesigns → click the LDD icon → done.

---

## ⚠️ Notes

- Don’t delete the extracted folder after installing  
- If UI doesn’t show → toggle extension off/on
